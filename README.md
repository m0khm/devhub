### Admin

| Variable | Description | Default |
| --- | --- | --- |
| `ADMIN_USER` | Username for admin login. | `admin` |
| `ADMIN_PASSWORD` | Password for admin login. | `admin` |
| `ADMIN_SESSION_TTL_MINUTES` | Admin session lifetime in minutes. | `60` |
